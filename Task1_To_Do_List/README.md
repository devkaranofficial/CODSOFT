# To-Do List Application

## Description

A simple command-line To-Do List application developed using Python. This project allows users to manage their daily tasks efficiently through a menu-driven interface.

## Features

* View Tasks
* Add New Tasks
* Update Existing Tasks
* Delete Tasks
* Input Validation
* Exception Handling
* User-Friendly Menu Interface

## Concepts Used

* Python Lists
* Loops
* Conditional Statements
* Functions
* Exception Handling (try-except)
* CRUD Operations

## How to Run

1. Clone the repository
2. Navigate to the project folder
3. Run:

python todo.py
