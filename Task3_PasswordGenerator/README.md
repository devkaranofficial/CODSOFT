# Password Generator using Python

A Python-based password generator that creates strong and random passwords using letters, numbers, and special characters.

## Features

- User-defined password length
- Random password generation
- Includes:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special symbols
- Input validation using try-except
- Generate multiple passwords
- Exit option

## Technologies Used

- Python
- random module
- string module
