# Contact Book

## Description

A Contact Book application developed using Python as part of the CodSoft Python Programming Internship.

This project allows users to manage contacts through a menu-driven interface. Users can add, display, search, update, and delete contact information efficiently.

## Features

* Add new contacts
* Display all saved contacts
* Search contacts by name
* Update existing contact details
* Delete contacts
* Phone number validation
* Case-insensitive search
* Menu-driven interface

## Technologies Used

* Python 3

## Concepts Used

* Variables
* User Input
* Conditional Statements
* Loops
* Functions
* Lists
* Dictionaries
* CRUD Operations
* Exception Handling
* Input Validation
* String Methods

## How to Run


   ```bash
   cd CODSOFT/Task5_Contact_Book
   ```

