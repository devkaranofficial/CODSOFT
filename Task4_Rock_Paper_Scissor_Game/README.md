# Rock Paper Scissors Game

## Description

A command-line Rock Paper Scissors game developed using Python as part of the CodSoft Python Programming Internship.

The game allows users to play multiple rounds against the computer, track scores, and view results after each round.

## Features

* Rock, Paper, Scissor gameplay
* Random computer choices using the random module
* Win, Lose, and Draw detection
* Score tracking system
* Input validation
* Exception handling using try-except
* Menu-driven interface

## Concepts Used

* Python Lists
* while Loops
* Conditional Statements
* random Module
* random.choice()
* Exception Handling
* Counters
* User Input Validation

## How to Run

1. Open terminal
2. Navigate to the project folder
3. Run:

python Rock_Paper_Scissors.py
